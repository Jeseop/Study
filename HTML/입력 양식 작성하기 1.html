form 태그 속성

  • method : 사용자가 입력한 내용을 어떻게 서버 쪽 프로그램으로 넘겨줄 지 지정. get방식과 post방식이 있음
  • name : 폼의 이름 지정
  • action : form 태그 안의 내용을 처리할 서버 프로그램 지정
  • target : 현재 창이 아닌 다른 위치에서 열도록 함

자동 완성 기능을 나타내는 autocomplete 속성이 있다. 기본 속성값은 on이고, off하면 기능 끄기 가능

form 요소를 그룹으로 묶는 fieldset과 legend 배우기

fieldset이란 form요소에서 연관된 요소들을 하나의 그룹(네모)으로 묶을 때 사용한다.
legend는 fieldset 태그의 캡션을 정의한다.
