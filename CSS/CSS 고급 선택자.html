1. 연결 선택자 하위 요소에 스타일을 적용하는 하위 선택자와 자식 선택자 특정 요소 기준 그 안에 포함된 요소를 '하위 요소' 현재 요소 기준 바로 한 단계 아래 요소 '자식 요소' 자식 요소의 한 단계 아래 '손자 요소' 하위 선택자 (=자손 선택자) 부모 요소에 포함된 하위 요소 모두 선택 상위 요소와 하위 요소를 '공백'으로 구분 기본 형태 : 상위요소 하위요소 자식 선택자 하위 선택자와 다르게 '자식요소에만' 스타일을 적용하는 서택자 두 요소 사이에 '>' 기호를 표시해 부모 요소와 자식 요소를 구분 [자식 선택자는 하위 선택자와 비슷해 보이지만 바로 한 단계 아래의 요소, 즉 '자식 요소에만' 스타일이 적용됨] 형제 요소에 스타일을 적용하는 인접 형제 선택자와 형제 선택자 웹 문서에서 부모 요소가 같을 경우 '형제 관계' 형제 관계인 요소에서 먼저 나오는 요소 '형 요소', 나중에 나오는 요소 '동생 요소' 인접 형제 선택자 형제 요소 중 첫 번째 동생 요소만 선택하는 것 요소와 요소 사이에 '+' 기호를 표시 형제 선택자 형제 요소의 스타일을 정의. 인접 형제 선택자와 달리 모든 형제 요소에 적용됨. 요소와 요소 사이에 '~' 기호를 표시 2. 속성 선택자 특정 속성이 있는 요소를 선택하는 [속성] 선택자 HTML 태그를 쓸 때 여러 가지 속성을 함께 사용하는데 그 속성값에 따라 원하는 요소를 선택할 수도 있음 이 때 사용하는 것이 [속성] 선택자. 대괄호 사이에 원하는 속성을 입력하면 됨. ex) a [href] {} 특정 속성값이 있는 요소를 선택하는 [속성 = 속성값] 선택자 주어진 속성과 속성값이 일치하는 요소를 찾아 스타일을 지정할 때 사용 ex) a [target = _blank] {} 여러 값 중에서 특정 속성값이 포함된 속성 요소를 선택하는 [속성 ~= "값"] 선택자 여러 속성값 중 해당 속성값이 포함된 요소를 선택함. 이 선택자는 속성이 하나면서 속성값이 여러 개일 때 특정 속성값을 찾는 데 편리함 ex) 하나의 요소에 클래스 스타일을 여러 개 적용할 수 잇는데, 그 중 button 스타일이 있는 요소를 찾으려면 [class ~= "button"] {} 과 같이 작성함 특정 속성값이 포함된 속성 요소를 선택하는 [속성 |= 값] 선택자 특정 속성값이 포함된 속성에 스타일을 적용함. 이때 속성값은 한 단어로 일치해야 함 [속성 ~= 값] 선택자와 비슷해 보이지만, [속성 |= 값] 선택자는 지정한 값과 '정확하게 일치'하거나 지정한 값을 포함해서 '하이픈(-)으로 연결된 단어'도 선택 ex) a[title |= us] {} 특정 속성값으로 '시작'하는 속성 요소를 선택하는 [속성 ^= 값] 선택자 속성값이 정확하게 일치하지 않아도 지정한 속성값으로 시작하는 요소를 찾으려면 [속성 ^= 값] 선택자를 사용 ex) a[title ^= eng] {} 특정한 값으로 끝나는 속성의 요소를 선택하는 [속성 $= 값] 선택자 지정한 속성값으로 끝나는 요소를 선택 ex) 링크한 파일 이름의 마지막 단어(확장자)가 xls인 요소를 찾을 때, [href $= xls] {} 일부 속성값이 일치하는 요소를 선택하는 [속성 *= 값] 선택자 속성값이 어느 위치에 있든지 지어한 속성값이 포함되어 있다면 해당 요소를 선택 ex) href 속성값 중 'w3'가 포함된 요소를 선택할 때, [href *= w3] {}
