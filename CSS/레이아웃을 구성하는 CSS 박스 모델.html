태그 처럼 문잔의 왼쪽이나 오른쪽에 이미지를 표시를 해야할 때가 있음. 그런데

태그는 블록 레벨 요소이니까 이미지가 나란히 한 줄에 배치하지 못함 이럴 때 float 속성을 사용하여 이미지를 표시하고 그 주변에 텍스트가 둘러 싸도록 할 수 있음 float 속성은 웹에 있는 요소를 문서 위에 떠 있게 만듬. (떠있다라는 건 요소가 왼쪽 구석이나 오른쪽 구성에 배치된다는 것) float 속성은 총 3개 left / right / none (좌우 어느쪽에도 배치하지 않음) 이 있음 float 속성을 해제하는 clear 속성 float 속성을 사용하면 웹의 요소를 왼쪽이나 오른쪽에 배치할 수 있음. 그런데 그 다음에 넣는 다른 요소에도 float 속성이 전달이 됨. 따라서 float 속성이 더 이상 필요없다고 알려주는 속성이 있음. 그것은 바로 clear 속성. clear 속성값 : left (float:left를 해제) / right (float:right를 해제) / both (float:left와 float:right를 해제)
